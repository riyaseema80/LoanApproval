# 🏦 Loan Approval Prediction App

A Machine Learning web application that predicts whether a loan will be approved or not based on user input.

---

## 🚀 Project Overview

This project uses **Logistic Regression** to solve a classification problem:

👉 Predict Loan Status:
- ✅ Approved  
- ❌ Not Approved  

The model is trained on real-world loan data and deployed using **Streamlit**.

---

## 🧠 ML Concepts Used

- Data Cleaning & Preprocessing  
- Handling Missing Values  
- Encoding Categorical Variables  
- Feature Scaling  
- Logistic Regression  
- Model Evaluation  

---

## 📊 Features Used

- Gender  
- Married  
- Dependents  
- Education  
- Self Employed  
- Applicant Income  
- Coapplicant Income  
- Loan Amount  
- Loan Term  
- Credit History  
- Property Area  

---

## 🛠️ Tech Stack
